In short I want to create a program that lets me extract midi values from my "focus level" while playing, in order to control effects in a daw. In effect I want to get as close to mind controled guitar pedals as I currently can with my skill and budget. I plan to use the Forcetrainer ii headset, as its very cheap, and can be connected to using some very well documented software made by the manufacturer. Then I need to be able to process this data and finally send it to a daw as midi messages that i can use to control effects.
